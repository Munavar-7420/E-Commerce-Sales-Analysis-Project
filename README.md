## 📌 Project Overview

This project analyzes an e-commerce retail dataset to uncover insights into sales performance, profitability, customer segments, and product categories. The analysis was performed using **Python (Pandas)** for data cleaning and **Power BI** for dashboard development.

---

## 🎯 Objectives

✔ Analyze sales and profit trends

✔ Identify top-performing products

✔ Evaluate regional sales performance

✔ Compare customer segments

✔ Generate business insights for decision-making
